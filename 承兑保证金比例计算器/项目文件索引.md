## 🔗 文件依赖关系

```
承兑保证金计算器_PC端.html
    ↓ (iframe嵌入)
FTP配置管理_PC端.html
    ↓ (localStorage共享)
承兑保证金计算器_移动端.html
```

**说明**:
- PC端计算器可以通过弹窗调用FTP配置页面
- 所有页面通过localStorage共享FTP配置数据
- 移动端和PC端独立运行，但共享相同的计算逻辑

## 💾 数据存储

### localStorage键值
- `userRole`: 用户角色（admin/user）
- `currentFTP`: 当前生效的FTP配置
- `ftpHistory`: FTP配置历史记录

### 数据格式
```json
{
  "ftpValue": 2.13,
  "effectiveDate": "2026-02-26",
  "remarks": "当前市场FTP",
  "operator": "当前用户",
  "operateTime": "2026-02-26T14:45:00.000Z",
  "status": "active"
}
```

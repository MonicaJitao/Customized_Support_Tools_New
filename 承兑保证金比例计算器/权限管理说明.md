# 承兑保证金比例计算器 - 权限管理说明

## 权限体系

### 角色定义

系统支持两种用户角色：

| 角色 | 标识 | 权限说明 |
|------|------|----------|
| 管理员 | `admin` | 可查看配置中心、修改FTP值、使用所有功能 |
| 普通用户 | `user` | 只能使用计算功能，不能修改FTP值 |

## 权限控制功能

### 管理员权限（admin）

#### 可见功能
- ✅ 右上角显示"⚙️ 配置中心"按钮
- ✅ 可以点击进入FTP配置管理页面
- ✅ FTP输入框可编辑（临时修改）
- ✅ 可以使用所有计算功能

#### 操作权限
- ✅ 配置FTP值
- ✅ 查看历史配置记录
- ✅ 导出配置数据
- ✅ 应用历史配置
- ✅ 删除历史记录

### 普通用户权限（user）

#### 可见功能
- ✅ 只能看到计算功能
- ❌ 看不到"配置中心"按钮
- ❌ FTP输入框禁用（灰色显示）
- ✅ 可以使用所有计算功能

#### 操作限制
- ❌ 无法修改FTP值
- ❌ 无法进入配置中心
- ❌ 无法查看历史配置
- ✅ 只能使用当前系统配置的FTP进行计算

## 权限设置方法

### 方法1: 通过JavaScript设置（推荐）

在页面加载后调用 `setUserRole()` 函数：

```javascript
// 设置为管理员
setUserRole('admin');

// 设置为普通用户
setUserRole('user');
```

### 方法2: 通过浏览器控制台设置（测试用）

打开浏览器开发者工具（F12），在控制台输入：

```javascript
// 设置为管理员
localStorage.setItem('userRole', 'admin');
location.reload();

// 设置为普通用户
localStorage.setItem('userRole', 'user');
location.reload();
```

### 方法3: 通过localStorage直接设置

```javascript
// 设置角色
localStorage.setItem('userRole', 'admin'); // 或 'user'

// 刷新页面使权限生效
window.location.reload();
```

## 集成到RM系统

### 嵌入方式

```html
<iframe id="calculatorFrame" src="path/to/承兑保证金计算器_PC端.html"></iframe>
```

### 权限传递

在RM系统中，根据当前登录用户的角色设置权限：

```javascript
// 获取RM系统当前用户角色
const rmUserRole = getCurrentUserRole(); // RM系统的函数

// 设置计算器权限
if (rmUserRole === '管理员' || rmUserRole === '风控人员') {
    localStorage.setItem('userRole', 'admin');
} else {
    localStorage.setItem('userRole', 'user');
}

// 加载计算器页面
document.getElementById('calculatorFrame').src = 'path/to/承兑保证金计算器_PC端.html';
```

### 跨iframe通信（可选）

如果需要动态切换权限，可以使用postMessage：

```javascript
// RM系统代码
const calculatorFrame = document.getElementById('calculatorFrame');

// 发送权限信息
calculatorFrame.contentWindow.postMessage({
    type: 'setUserRole',
    role: 'admin' // 或 'user'
}, '*');
```

在计算器页面添加监听：

```javascript
// 计算器页面代码
window.addEventListener('message', function(event) {
    if (event.data.type === 'setUserRole') {
        setUserRole(event.data.role);
    }
});
```

## 权限验证流程

```
用户访问页面
    ↓
检查localStorage中的userRole
    ↓
    ├─ admin → 显示配置中心按钮 + FTP可编辑
    └─ user/null → 隐藏配置中心按钮 + FTP禁用
    ↓
加载当前FTP配置
    ↓
用户使用计算功能
```

## 安全说明

### 当前实现
- 权限信息存储在浏览器localStorage
- 前端控制界面显示和输入框状态
- 适用于内部系统，信任用户环境

### 安全建议

#### 生产环境建议
1. **后端验证**: 将FTP配置存储在后端，通过API获取
2. **身份认证**: 集成RM系统的SSO单点登录
3. **权限校验**: 每次操作都在后端验证用户权限
4. **审计日志**: 记录所有FTP配置修改操作

#### 增强安全性（可选实现）

```javascript
// 示例：通过API验证权限
async function checkUserPermission() {
    try {
        const response = await fetch('/api/user/permission', {
            headers: {
                'Authorization': 'Bearer ' + getToken()
            }
        });
        const data = await response.json();

        if (data.role === 'admin') {
            document.getElementById('configBtn').style.display = 'block';
        }
    } catch (error) {
        console.error('权限验证失败', error);
    }
}
```

## 常见问题

### Q1: 如何查看当前用户角色？
```javascript
// 在浏览器控制台输入
getUserRole();
// 返回 'admin' 或 'user'
```

### Q2: 权限设置后不生效？
**解决方案**:
1. 检查localStorage是否正确设置
2. 刷新页面
3. 清除浏览器缓存

### Q3: 如何重置权限？
```javascript
// 清除权限设置
localStorage.removeItem('userRole');
location.reload();
// 默认为普通用户
```

### Q4: 管理员临时修改的FTP会保存吗？
**答**: 不会。管理员在计算页面临时修改的FTP只在当前计算中生效，不会保存。只有通过"配置中心"修改的FTP才会持久化保存。

### Q5: 多个用户同时使用会冲突吗？
**答**: 不会。每个用户的权限和配置都存储在各自的浏览器localStorage中，互不影响。

```javascript
const ROLES = {

function checkUserPermission() {
    const role = getUserRole();

    switch(role) {

            // 所有权限

            break;
        case ROLES.USER:
        default:
            // 只能计算
            break;
    }
}
```

## 技术实现

### 核心代码

```javascript
// 权限检查
function checkUserPermission() {
    const userRole = localStorage.getItem('userRole');

    if (userRole === 'admin') {
        // 管理员权限
        document.getElementById('configBtn').style.display = 'block';
        document.getElementById('ftp1').disabled = false;
        document.getElementById('ftp2').disabled = false;
    } else {
        // 普通用户权限
        document.getElementById('configBtn').style.display = 'none';
        document.getElementById('ftp1').disabled = true;
        document.getElementById('ftp2').disabled = true;
    }
}

// 设置用户角色
function setUserRole(role) {
    localStorage.setItem('userRole', role);
    checkUserPermission();
    loadFTPConfig();
}
```

# 用户权限配置说明

## 权限配置存档

### 当前权限设置

通过 `设置管理员权限.html` 页面设置的权限会自动保存到浏览器的 localStorage 中。

### 存储位置

- **存储键**: `userRole`
- **存储值**: `admin` 或 `user`
- **存储位置**: 浏览器 localStorage

### 查看当前权限

打开浏览器开发者工具（F12），在控制台输入：

```javascript
localStorage.getItem('userRole');
```

返回值：
- `"admin"` - 管理员权限
- `"user"` - 普通用户权限
- `null` - 未设置权限

### 权限持久化

权限设置会持久化保存，除非：
1. 手动清除浏览器数据
2. 使用"清除权限"按钮
3. 手动删除 localStorage

### 导出权限配置

如需备份权限配置，可在控制台执行：

```javascript
// 导出当前权限配置
const config = {
    userRole: localStorage.getItem('userRole'),
    exportTime: new Date().toISOString()
};
console.log(JSON.stringify(config, null, 2));
```

### 导入权限配置

```javascript
// 导入权限配置
localStorage.setItem('userRole', 'admin'); // 或 'user'
location.reload();
```

## 权限配置文件

如需在多台设备间同步权限，可以：

1. 在设置页面设置权限
2. 权限自动保存到 localStorage
3. 在其他设备上重新设置相同权限

## 注意事项

- 权限配置仅在当前浏览器有效
- 不同浏览器需要分别设置
- 清除浏览器数据会丢失权限配置
- 建议记录当前使用的权限角色

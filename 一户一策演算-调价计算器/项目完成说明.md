### 4. 计算逻辑 ✅

#### 兜底价格计算
```javascript
if (客户类型 === 'Y') {
    兜底价格 = 业务成本 + 当前风险成本 + (计提利润点 / 100)
} else {
    兜底价格 = 非520客户兜底价格
}
```

#### 核心公式
```
对应3个月长周期目标余额 = (当前价格 - 兜底价格) × 近3个月长周期余额 / (目标价格 - 兜底价格)
日均需提额余额 = 对应3个月长周期目标余额 - 近3个月长周期余额
对应用款天数目标余额 = (预计用款天数 / 90) × 对应3个月长周期目标余额
```

### 7. 使用流程 ✅

#### 首次使用
1. 打开 `设置管理员权限.html`
2. 选择"管理员"角色
3. 点击"保存设置"
4. 打开 `调价计算器_PC端.html`
5. 点击右上角"参数配置中心"
6. 配置业务参数（或使用默认值）
7. 返回计算器进行计算

#### 日常使用
1. 打开计算器（移动端或PC端）
2. 选择客户类型
3. 输入7个参数
4. 点击"计算目标余额"
5. 查看三个结果和曲线图

### 10. 业务场景示例 ✅

#### 场景1: 520客户降价
- 当前价格: 7.99%
- 风险成本: 0.5%
- 近3个月余额: 205万元
- 目标价格: 6.00%

## 已知限制

1. **数据存储**: 使用 localStorage，清除浏览器数据会丢失配置
2. **跨浏览器**: 不同浏览器的 localStorage 不共享
3. **离线使用**: 需要先访问一次页面，之后可离线使用
4. **大数值**: 当计算结果过大（>10000万元）时，建议重新评估参数
